### Задание 
#### Написать функцию, которая находит самое часто встречающееся число в каждой строке матрицы и возвращает массив этих значений

### Импорт


```python
from statistics import mode
import numpy as np
```

### Считаем моду в каждой строке


```python
def f(x):
    modes = []
    for line in x:
        modes.append(mode(line))
    return modes
```
